# Node Shoes Shop

Monorepo scaffold for microservices (auth, order, user) + gateway and shared libs.

Mọi service đều là TypeScript Express app tối giản trong `services/*/src`.

## Chạy toàn bộ service

```bash
npm install
npm run dev
```

## Chạy bằng Docker Compose

```bash
docker-compose up --build
```
